📖 How to get a good PNG

1. Place maps in frames

Arrange all item frames on a flat wall matching your grid dimensions.

2. Use Optifine zoom

Hold C to zoom in and capture the full mapart without distortion.

3. Screenshot (F2)

Press F2 in Minecraft. Screenshots are saved to .minecraft/screenshots.

4. Crop and upload

Optionally crop out the HUD. Then upload the PNG here.

ℹ About This Archive

Welcome to the 6b6t Mapart Archive

This is a community-driven gallery dedicated to preserving and showcasing Minecraft map art created on the 6b6t.org anarchy server. Map art is one of the most creative and impressive forms of expression in Minecraft — painstakingly built block-by-block and rendered as pixel art across dozens of in-game maps.

What is Mapart?

Mapart (map art) is created by arranging dyed blocks at a specific Y level underground so that, when held in item frames, the maps display a pixel art image. A single "map" covers a 128×128 block area, so a 10×9 piece uses 90 maps and spans 1280×1152 blocks!

About 6b6t

6b6t.org is one of the longest-running Minecraft anarchy servers — a lawless world where players build, destroy, and create freely. Despite the chaos, a thriving mapart community has grown here, producing some of the most stunning pixel art in the game.

How to Contribute

Anyone can submit their mapart using the Upload page. Your creation will appear immediately in the gallery. For permanent storage across all users and devices, the data/maparts.json file should be updated in the GitHub repository.

Technical Notes

This site is 100% static HTML/CSS/JS — no server required. It runs on GitHub Pages. Upload data is stored in your browser's localStorage. To add entries permanently, edit data/maparts.json and commit to the repository.
